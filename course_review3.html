Selecting Programming Language


C/C++의 hardware related feature
: ① Physical memory management
물리적인 컴퓨터 메모리, RAM을 직접 제어 가능
포인터: 변수의 값이 저장된 컴퓨터 메모리의 주소를 저장
Memory mapped I/O: CPU가 입출력을 처리할 때 컴퓨터 메모리의 일정 영역에 직접 값을 읽고 씀(입출력 데이터를 위한 별도의 주소 공간을 두지 x),
메모리의 원하는 위치에 접근하기 위해 포인터 변수 사용
② Bit Operation: 0과 1의 비트를 처리 가능

Microservice: Cloud Computing에 기반하여 프로그램, 서버를 여러 개로 잘게 쪼갬
Polyglot: 마이크로서비스로 프로그램을 쪼개서 각자 개발
-> 부서마다 언어를 통일할 필요 X, 다양한 목적에 맞는 언어로 개발 후 통신으로 연결

C++
: general purpose, low-level, compiled language
유일한 ISO 산업 표준 언어
Python
: high-level, interpreted language
문법이 쉬움, 라이브러리 많으므로 코드 짧음, 개발 기간 짧음
dynamic: type 정의 x, type 다른 변수 간 연산 가능
glue language: 다른 언어로 작성한 코드 이어붙이기 적합

+) Compiler vs Interpreter
Compiler: 코드 전체를 스캔 후 한 번에 번역하여 실행 파일 생성
Interpreter: 코드 실행 시 한 번에 한 줄씩 번역하여 실행(line by line)
